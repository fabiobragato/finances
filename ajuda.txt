---------------------- Comandos ------------------------ 

---------------------- Relatórios da AWS ----------------------

buscar_relatorio(conta, ano, trimestre): 
- Objetivo: busca os dados das demonstrações financeiras de conglomerados financeiros. 
- Primeiro Parâmetro (conta): informar o relatório desejado (ativo, passivo, DRE). 
- Segundo Parâmetro (ano): informar o ano desejado. 
- Terceiro Parâmetro (trimestre): informar o trimestre desejado. 

buscar_relatorios(conta, ano_inicial, trimestre_inicial, ano_final, trimestre_final): 
- Objetivo: busca os dados das demonstrações financeiras de conglomerados financeiros no intervalo determinado. 
- Primeiro Parâmetro (conta): informar o relatório desejado (ativo, passivo, DRE). 
- Segundo Parâmetro (ano_inicial): informar o ano inicial. 
- Terceiro Parâmetro (trimestre_inicial): informar o trimestre inicial desejado. 
- Quarto Parâmetro (ano_final): informar o ano final. 
- Quinto Parâmetro (trimestre_final): informar o trimestre final desejado. 

ultimo_relatorio_disponivel(conta): 
- Objetivo: traz o relatório disponível mais recente. 
- Parâmetro (conta): informar o relatório desejado (ativo, passivo, DRE).

---------------------- Relatórios Web Scraping ----------------------

buscar_balancetes(safra_inicial, safra_final): 
- Objetivo: busca os dados dos balancetes divulgados pelo Bacen das Instituições Financeiras. 
- Primeiro Parâmetro: informar a safra inicial desejada no formato anomês, ex: 202001. 
- Segundo Parâmetro: informar a safra final desejada no formato anomês, ex: 202004. 

---------------------- Relatórios Yahoo Finance ----------------------

dados_acoes_yahoo(tickers, data_inicial, data_final, tipo): 
- Objetivo: traz em formato de tabela o histórico das ações do tipo selecionado. 
- Primeiro Parâmetro: código da ação. Exemplo: MGLU3.SA. 
- Segundo Parâmetro: data inicial no formato "ano-mes-dia", ex: "2020-01-01". 
- Terceiro Parâmetro: data final no formato "ano-mes-dia", ex: "2020-02-10". 
- Quarto Parâmetro: tipo do relatório ("G", "F", "C", "H", "L", "O" ou "V"), no qual: 
        G: Todas as informações disponíveis 
        F: Preço de Fechamentos Ajustados 
        C: Preços de Fechamento 
        H: Preços Máximos 
        L: Preços Mínimos 
        O: Preços de Abertura 
        V: Volume de Negociações 

plotar_grafico_fechamentos(tickers, data_inicial, data_final): 
- Objetivo: plota o gráfico de fechamento das ações selecionadas na base 100. 
- Primeiro Parâmetro: código da ação. Exemplo: MGLU3.SA. 
- Segundo Parâmetro: data inicial no formato "ano-mes-dia", ex: "2020-01-01". 
- Terceiro Parâmetro: data final no formato "ano-mes-dia", ex: "2020-02-10". 

retornos_simples(tickers, data_inicial, data_final): 
- Objetivo: traz em formato de tabela os retornos simples das ações selecionadas. 
- Primeiro Parâmetro: código da ação. Exemplo: MGLU3.SA. 
- Segundo Parâmetro: data inicial no formato "ano-mes-dia", ex: "2020-01-01". 
- Terceiro Parâmetro: data final no formato "ano-mes-dia", ex: "2020-02-10". 

retornos_log(tickers, data_inicial, data_final): 
- Objetivo: traz em formato de tabela os retornos logarítmicos das ações selecionadas. 
- Primeiro Parâmetro: código da ação. Exemplo: MGLU3.SA. 
- Segundo Parâmetro: data inicial no formato "ano-mes-dia", ex: "2020-01-01". 
- Terceiro Parâmetro: data final no formato "ano-mes-dia", ex: "2020-02-10".
